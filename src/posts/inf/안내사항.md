## 05. 퍼블리싱 작업 권고사항
* anchor & button 태그 사용 기준
    - `<a>`태그는 페이지 이동인 경우 사용 
    - `<button>` 태그는 동작이나 상태 변경인 경우 사용
* 들여쓰기는 **space 4칸** 기준으로 합니다.
```html
<div class="c_confirm_button_box">
    <button class="button_cancel">이전</button>
    <button class="button_confirm">다음</button>
</div>
```
* 의미없는 줄바꿈,공백은 지양합니다.
```html
<!-- (잘못된 예) -->
  <p>
      신한  쏠비즈   기업뱅킹   
  </p>

<!-- (올바른 예) -->
  <p>신한 쏠비즈 기업뱅킹</p>
```
* 구분을 위한 줄바꿈은 한줄만 허용합니다.
```html
  <div class="t_option_col_group">
      // 생략
  </div>

  <div class="t_option_row_group">
      // 생략
  </div>
```
* 단일 태그는 닫기('/') 없이 사용합니다.
```html
  <br/> X / <br> O
```
* css에서 **color** 관련 속성을 사용할 경우 변수 컬러값을 사용합니다.
    - 관련 css 파일 : variable.css
```css
.c_result.border{border-top: 0; border-bottom: 1px solid var(--bc-1);}
```

## 06. class 규칙
* **BEM**(Block Element Modifier) 네이밍 규칙을 따릅니다.
    - 단어와 단어 사이 (_) 로 구분
```html
<div class="button_box"></div>
```
* 클래스명 순서는 '종류 &rarr; 중분류 &rarr; 형태' 준수하여 작업합니다.
```html
<div class="c_accordion_question_box"></div>
```
* 공통클래스를 제외한 약어는 사용 지양합니다.
```html
<h3 class="s_tit"></h3> - X
<h3 class="sub_title"></h3> - O
```
* 상태값은 "is_" 접두어를 붙여 사용합니다.
```html
<div class="c_accordion is_open"></div>
```
* 구분을 위한 접두어는 앞에 붙여 사용합니다. ( 공통 접두어 사용 )
```html
template  인 경우 (t_) : <div class="t_name"></div>
component 인 경우 (c_) : <div class="c_name"></div>
element   인 경우 (e_) : <i class="e_name"></i>
```
* 공통 간격과 다른 경우 공통클래스를 사용하여 조정합니다.
    - 공통클래스 관련 파일 : common.css
```html
<div class="c_title_heading h4 mt32"> <!-- ex: 상단간격 32 -> 'mt32' 클래스 사용 -->
    <div class="title_box">
        <strong class="title">자산현황</strong>
    </div>
</div>
```

## 07. javascript 작성 규칙
* **카멜 표기법**(camel Case) 네이밍 규칙을 따릅니다.
    - 처음 단어는 소문자 표기, 그 다음 단어부터 첫글자 대문자 표기
&nbsp;&nbsp;&nbsp;
&nbsp;&nbsp;&nbsp;
```js
    function init() {
        setDom()
        setInput()
    }
```
* 변수명 지정할때 (_)로 시작해서 사용하는 것을 권장합니다.
```js
    var _self = this
    var _dom = null
```
* UI 스크립트의 실행을 위한 선택 요소로, 클래스명 대신 데이터 속성값을 사용하는 것을 권장합니다.
```html
  <div class="c_tab_menu" data-module="tabMenu" data-target="addrPopTab-A">
    // 생략
  </div>
```
```js
    function setTabMenu() {
        var _items = document.querySelectorAll("[data-module='tabMenu']")
        for (var i = 0; i < _items.length; i++) {
            var __item = _items[i]
            var __tabTarget = __item.getAttribute("data-target")
            if (!__item.classList.contains("initiate")) {
                __item.module = new UIComponent.tabMenu({
                    'dom': __item,
                    'target': __tabTarget
                })
                __item.classList.add("initiate")
            }
        }
```
